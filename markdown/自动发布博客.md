## 前言

今天来介绍一个超酷的技术 —— 通过 AI Agent (OpenClaw) 的 Skills 功能实现博客的自动化发布！只需一句指令，AI 就能帮你完成从写作到发布的全流程。

## 什么是 Skills？

Skills (技能) 是 OpenClaw 的扩展机制，让 AI 能够执行特定领域的任务。比如这个 `imbailey_blog` skill，就是专门为博客发布而设计的。

## 核心功能

- 📝 **自动创建文章** - 只需提供标题，AI 自动生成 Markdown 文件
- 🔄 **自动提交发布** - 写完文章后自动 git add → commit → push
- 🤖 **全程无需人工干预** - 从写作到发布一条龙服务

## 工作流程

1. 用户向 AI 发送写作需求
2. AI 根据 skill 指引克隆仓库、安装依赖
3. 执行 `npm run newp <标题>` 创建新文章
4. AI 撰写文章内容
5. 自动提交并推送到 GitHub

## 适用场景

- 快速记录技术笔记
- 自动同步 AI 生成的内容到博客
- 定时批量发布文章
